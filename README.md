# TerosHDL

## Dependencies

- Symbolator:
```pip install symbolator```
- TerosHDLbackend:
```pip install TerosHDL```
- VUnit:
```pip install vunit_hdl```

For simulation:

- Ghdl/Modelsim

For code coverage:

- Ghdl with GCC backend.

For waveform:

- GTKWave/ModelSim

For structure-view:

- Ctags

## Installation

```apm install terosHDL```
