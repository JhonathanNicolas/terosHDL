# TerosHDL

## Dependencies

- Symbolator:
```sudo pip install symbolator```
- TerosHDLbackend:
```sudo pip install TerosHDL```

For simulation:

- Ghdl/Modelsim

For code coverage:

- ghdl with gcc backend.

For waveform:

- gtkwave.

For structure-view:

- ctags

## Installation

- From source code:
```apm link .```
```apm install```
- From atom:
Search package: TerosHDL
